# Overview

The goal of this project was to create a 3D Maze Escape Game to improve my skills as a software engineer in game development using GDScript. I wanted to explore 3D environments, player movement, and game logic while making an interactive and fun experience.  

In the game, the player controls a first-person character to navigate through a maze. The main objective is to find a key and unlock the exit door. The player can move using the WASD keys, look around with the mouse, and jump over small obstacles in the maze. Some levels include narrow paths, dead ends, and minor parkour sections to increase the challenge.  

The purpose of creating this software was to learn how to implement game mechanics, handle user input, design levels, and manage interactive objects in a 3D environment. It also helped me practice debugging, problem-solving, and applying physics-based movement in Godot.
